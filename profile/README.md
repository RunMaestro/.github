> *Maestro hones fractured attention into focused intent.*

Maestro is a cross-platform desktop app for orchestrating your fleet of AI agents and projects. It's a high-velocity solution for hackers who juggle multiple projects in parallel — designed for power users who live on the keyboard and rarely touch the mouse.

Currently supporting **Claude Code**, **OpenAI Codex**, **OpenCode**, and **Factory Droid**.

## Highlights

- **Multi-Agent Management** — Run unlimited agents in parallel, each with isolated context and conversation history
- **Auto Run & Playbooks** — Batch-process markdown checklists through AI agents for long-running unattended sessions
- **Git Worktrees** — Run AI agents on isolated branches with true parallel development
- **Group Chat** — Coordinate multiple AI agents in a single conversation with an AI moderator
- **Mobile Remote Control** — Monitor and control all agents from your phone via built-in web server
- **CLI** — Full `maestro-cli` for headless operation, cron jobs, and CI/CD pipelines
- **Keyboard-First** — Full keyboard control with customizable shortcuts and mastery tracking

## Quick Start

Download the [latest release](https://github.com/RunMaestro/Maestro/releases) for macOS, Windows, or Linux.

```bash
# Or build from source
git clone https://github.com/RunMaestro/Maestro.git
cd Maestro && npm install && npm run dev
```

**Requires** at least one AI coding agent installed: [Claude Code](https://docs.anthropic.com/en/docs/claude-code), [OpenAI Codex](https://github.com/openai/codex), or [OpenCode](https://github.com/sst/opencode).
